# Narrative Prosecution

## Objective
Quantify the gap between what management SAYS and what the NUMBERS show.

## Claim Extraction Rules
1. Only extract SPECIFIC, MEASURABLE claims
   - GOOD: "We expect revenue to cross Rs 5000 Cr by FY26"
   - BAD: "We are optimistic about growth" (too vague)
2. Tag each claim with: category, target metric, target value, timeline
3. Minimum 5 claims per annual report, 3 per transcript
4. Record exact page/timestamp for traceability

## Promise-vs-Delivery Scoring
- **Delivered**: Actual meets or exceeds target within timeline
- **Partially Delivered**: Within 20% of target
- **Missed**: More than 20% below target
- **Quietly Dropped**: Theme disappears from subsequent filings

## Quietly Dropped Detection
The most important forensic signal. When management stops talking
about a strategy without explaining why:
1. Track theme presence across consecutive filings
2. If theme appears 2+ times then vanishes for 2+ quarters → FLAG
3. Automatic -3% Pattern Premium deduction per dropped theme
